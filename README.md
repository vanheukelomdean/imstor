# imstor

An image repository web application for the Shopify Backend Developer Internship challenge

### Usage

https://imstor.herokuapp.com/

### Features
 - Authentication
 - Multi-Image Uploads
 - Basic Privacy 
      - Access control with owner/world permissions
 - Permanent Storage 
      - AWS S3 file store
      - MongoDB nosql records)
